# arxiv_digest
Retrieve and format submissions from [arXiv.org](https://arxiv.org/) into an html file.

### What it does
The purpose of this script is to simplify the process of keeping up-to-date with submissions from [arXiv.org](https://arxiv.org/).
Specifically, `arxiv_digest` tracks the newly uploaded papers to the archives of interest and formats them into a convenient, easily-read html file, that can be opened in a browser or sent via email.
For every paper of interest the script lists the title (that is also a link to the full pdf), tags, and authors:

### How to run it
- **Prerequisites.** Install the requirements: `pip install -r requirements.txt`.
- **Customization.** Edit `settings.ini` to configure which archives to track, date range, keywords, display options, and email delivery.
- **Email setup.** To enable email delivery:
  1. Set `enabled = true` in the `[email]` section of `settings.ini` and fill in the SMTP host, port, from, and to addresses.
  2. Copy `.env.example` to `.env` and fill in your SMTP credentials (`SMTP_USER`, `SMTP_PASSWORD`).
  3. For Gmail, use an [App Password](https://myaccount.google.com/apppasswords).
- **Execution.** Run `python arxiv_digest.py` to fetch submissions, generate an html file in `html_files/`, and optionally send it via email.
- **Automation.** Set up a cron job to run the script regularly:
    ```bash
    # retrieve and email arxiv submissions every weekday at 7:00am est
    00 11 * * 1-5 cd /path/to/arxiv_digest && python arxiv_digest.py
    ```
